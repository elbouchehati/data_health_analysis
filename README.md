# Data Health Analysis

A data analysis project focused on health insurance data.

## Description

This project analyzes health insurance data to uncover insights and patterns that can help understand factors affecting insurance costs and coverage.

## Project Structure
```
data_health_analysis/
│
├── data/           # Data files
├── notebooks/      # Jupyter notebooks for analysis
├── src/            # Source code
├── tests/          # Unit tests
├── requirements.txt
└── README.md
```

## Installation

1. Clone this repository:
```bash
git clone https://github.com/elbouchehati/data_health_analysis.git
cd data_health_analysis
```

2. Install required packages:
```bash
pip install -r requirements.txt
```
